# 前端接口调用文档

## 一、用户信息类
### 1、登陆
- *调用路径`/api/auth/login`*
- *调用格式：*
```http request
POST https://mushu.onlion/api/auth/login HTTP/1.1

Content-Type: application/json

{
    id:xx
    password:xx
}
```
- *传入参数：* 
  - 1、id，类型 **int** ,用户id即学号
  - 2、password，类型 **String** ，用户输入的密码明文传递
- *返回格式：*

```json
{
  "id":0,
  "username":null,
  "code":null,
  "jwt":null
}
```
- *返回参数*
  - 1、id，类型 **int** ,用户id即学号
  - 2、username，类型 **String** ，用户名
  - 3、code，类型 **int** ，返回代码
  - 4、jwt，类型 **String** ，jwt令牌
    
| 返回代码 | 说明        | 
|------|-----------|
| 200  | 登陆成功，有jwt |
| 201  | 登录失败，无jwt |
   
### 2、注册

- *调用路径`/api/user/register`*
- *调用格式：*
```http request
POST https://mushu.onlion/api/auth/register HTTP/1.1

Content-Type: application/json

{
    id:xx
    password:xx
}
```
- *传入参数：*
  - 1、id，类型 **int** ,用户id即学号
  - 2、password，类型 **String** ，用户输入的密码明文传递
- *返回格式：*

```json
{
  "id":0,
  "code":null
}
```
- *返回参数*
- 1、id，类型 **int** ,用户id即学号
- 2、code，类型 **int** ，返回代码


| 返回代码 | 说明   | 
|------|------|
| 200  | 注册成功 |
| 201  | 注册失败 |

### 3、修改信息

- *调用路径`/api/user/reviseBaseInf`*
- *调用格式：*
```http request
POST https://mushu.onlion/api/auth/register HTTP/1.1

Content-Type: application/json

{
  "id": 0,
  "college": "",
  "grade": "",
  "telephone": "",
  "profile": "",
  "certified": false
}
```
- *传入参数：*
  - 1、id，类型 **int** ,用户id即学号
  - 2、college，类型 **String** ，学院
  - 3、grade，类型 **String** ，年级
  - 4、telephone，类型 **String** ，联系方式
  - 5、profile，类型 **String** ，个人信息
  - 6、certified，类型 **String** ，认证信息
- *返回格式：*

```json
{
  "id": 0,
  "college": "",
  "grade": "",
  "telephone": "",
  "profile": "",
  "certified": false
}
```
- *返回参数*
- 1、id，类型 **int** ,用户id即学号
- 2、college，类型 **String** ，学院
- 3、grade，类型 **String** ，年级
- 4、telephone，类型 **String** ，联系方式
- 5、profile，类型 **String** ，个人信息
- 6、certified，类型 **String** ，认证信息
