## 一、User接口
### 1、登陆
- *调用路径`/api/user/login`*
- *调用格式：*
```http request
POST https://mushu.onlion/api/user/login HTTP/1.1

Content-Type: application/json

{
    id:xx
    password:xx
}
```
- *传入参数：* 
  - 1、id，类型 **int** ,用户id即学号
  - 2、password，类型 **String** ，用户输入的密码明文传递
- *返回格式：*

```json
{
  "id":0,
  "username":null,
  "access": 0,
  "code":null,
  "jwt":null
}
```
- *返回参数*
  - 1、id，类型 **int** ,用户id即学号
  - 2、username，类型 **String** ，用户名
  - 3、code，类型 **int** ，返回代码
  - 4、access 类型 **int** ,用户权限
  - 5、jwt，类型 **String** ，jwt令牌
    
| 返回代码 | 说明        | 
|------|-----------|
| 200  | 登陆成功，有jwt |
| 201  | 登录失败，无jwt |
